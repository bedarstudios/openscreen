Empty states — centered icon in an accent-100 circle (accent for library/recording, accent-2 for the doc step), one serif heading, one muted sentence naming what will appear, one primary button naming the action. Never more than this.

Voice
Buttons name a verb and a noun — Start Recording, Create Workflow Doc — never a bare "OK."
Toasts confirm what happened, no "successfully" — Copied folder path, not "Successfully copied."
Empty states point to the next action — "No recordings yet. Start one to see it here."
In-progress uses a present participle and an ellipsis — "Generating your workflow doc…"
Title Case for buttons and nav; sentence case everywhere else.
Do / Don't
Do keep paper cream and ink near-black — never pure white or pure black.
Do pair the recording pulse with a text label; color alone never signals state.
Don't mix light and dark theme tokens in one view.
Don't use Accent 2 outside the Desktop source tag.
Don't mix sharp and round corners in the same surface.
